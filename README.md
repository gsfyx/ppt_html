# PPT 模板编辑器 (`ppt_template.html`)

## 概述

一个功能完善的单页 HTML 幻灯片编辑器，支持多主题切换、幻灯片管理、元素编辑、动画效果、全屏播放等功能。数据通过 `localStorage` 持久化缓存，支持 JSON/HTML 格式的导入导出。

## 主要功能

### 1. 多主题切换

提供 5 种内置视觉主题，一键切换全部幻灯片的配色和样式：

| 主题 | 类名 | 说明 |
|------|------|------|
| 党建红 | `theme-red` | 红色主色调，适用于党建、政务场景 |
| 科技蓝 | `theme-blue` | 蓝色主色调，适用于科技、企业场景 |
| 紫韵 | `theme-purple` | 紫色主色调，适用于创意、艺术场景 |
| 简约 | `theme-minimal` | 极简风格，适用于商务、汇报场景 |
| 办公 | `theme-office` | 经典办公风格，适用于日常办公场景 |

### 2. 幻灯片管理

- **新建幻灯片**：在当前幻灯片之后插入新页
- **复制幻灯片**：复制当前选中的幻灯片
- **删除幻灯片**：支持单页删除（带确认对话框）
- **拖拽排序**：通过左侧缩略图面板拖拽调整幻灯片顺序
- **缩略图面板**：左侧可折叠的缩略图导航栏，点击缩略图快速跳转

### 3. 幻灯片切换与转场

- 支持多种转场动画（淡入、滑入、缩放等）
- 可配置转场持续时间（默认 500ms）
- 支持自动播放模式，可配置自动切换间隔
- 上一页/下一页导航
- 全屏播放模式：隐藏工具栏和缩略图面板

### 4. 元素编辑功能

#### 4.1 文本编辑
- **双击编辑**：双击任意文本元素进入编辑模式
- **富文本格式**：支持粗体、斜体、下划线、删除线、字体大小
- **字体选择**：支持多种中英文字体
- **文字颜色**：支持文字颜色和背景色设置
- **对齐方式**：左对齐、居中、右对齐、两端对齐
- **超链接**：为选中元素添加可点击的链接
- **批注/评论**：为选中元素添加备注信息

#### 4.2 元素操作
- **拖拽移动**：任意元素可拖拽调整位置
- **缩放调整**：拖拽元素边角调整大小（带最小尺寸限制）
- **旋转**：支持元素旋转
- **框选多选**：鼠标拖拽框选多个元素
- **复制粘贴**：支持 Ctrl+C / Ctrl+V 复制粘贴元素
- **删除元素**：选中后按 Delete 或 Backspace 删除

#### 4.3 元素动画
- **入场动画**：支持左滑入、右滑入、淡入、缩放、弹入等效果
- **动画排序**：同一幻灯片内多个元素可指定动画播放顺序
- **动画配置**：支持设置持续时间、延迟、播放次数
- **启动模式**：支持单击触发、自动播放等模式

### 5. 插入功能

- **插入表格**：指定行列数创建表格，支持单元格编辑
- **插入图片**：从本地或通过 URL 插入图片
- **插入图标**：内置图标库，支持 SVG 图标插入
- **插入文本框**：创建可编辑的文本区域
- **插入形状**：支持矩形、圆形、三角形等基础形状
- **插入链接**：为选中内容添加超链接
- **插入批注**：为选中内容添加批注

### 6. 幻灯片背景

- **背景图片**：支持为当前页或全部页面设置背景图片
- **背景清除**：清除当前页或所有页面的背景图片
- **应用范围切换**：单页模式 vs 全局模式
- **平铺/拉伸**：背景图片自适应显示

### 7. 数据持久化

- **自动缓存**：所有编辑内容自动保存到 `localStorage`
- **缓存 Key**：以文件名作为缓存键，支持多文档管理
- **撤销/重做**：完整的撤销/重做历史栈（最多 120 步）
- **刷新恢复**：页面刷新后自动恢复上次编辑状态

### 8. 导入导出

- **导出为 JSON**：将完整幻灯片数据（含样式、元素、动画配置）导出为 JSON 文件
- **导出为 HTML**：导出为独立的 HTML 文件
- **从 JSON 导入**：加载 JSON 文件恢复幻灯片数据
- **缓存清除**：支持一键清空当前文档缓存

### 9. 底部导航栏

- 页码指示器：显示当前页 / 总页数
- 上一页/下一页按钮
- 全屏切换按钮
- 自动播放开关

### 10. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 上一页 / 下一页 |
| `Home` / `End` | 跳到首页 / 末页 |
| `Delete` / `Backspace` | 删除选中元素 |
| `Ctrl+C` | 复制选中元素 |
| `Ctrl+V` | 粘贴元素 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `F5` / `F` | 全屏播放 |
| `Esc` | 退出全屏 |

## 技术实现

- **纯前端**：无后端依赖，单 HTML 文件即可运行
- **CSS 变量体系**：通过 CSS 自定义属性实现主题统一切换
- **localStorage 缓存**：数据仅存储在浏览器本地
- **响应式布局**：适配不同屏幕尺寸
- **触摸支持**：支持触摸屏设备的基础操作

## 使用场景

- 在 WPS 插件中嵌入，作为 PPT 模板预览和编辑工具
- 独立使用，作为轻量级在线幻灯片编辑器
- 与其他系统集成时作为 PPT 数据结构的可视化编辑前端
